# OpenClaw Expenses

A personal expense tracking application built with a FastAPI backend and a Vue3 frontend. Designed for easy deployment and personal finance management.

## Features

*   **Expense Recording:** Quickly add new expense records with details like amount, type, and date.
*   **Categorization:** Organize expenses into custom categories for better analysis.
*   **Data Visualization:** (Coming Soon) View expense summaries and trends through charts and graphs.
*   **User Management:** Simple user system to keep your financial data private.

## Tech Stack

*   **Backend:** FastAPI (Python 3.9)
*   **Frontend:** Vue.js 3 (Node.js 18)
*   **Database:** MySQL
*   **Web Server:** Nginx
